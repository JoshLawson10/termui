termui.widget

Widget 

Widget(**kwargs)

Bases: DOMNode, ABC

Base class for all widgets in the TermUI framework.

Widgets are visual components that can be rendered to the terminal. They inherit from DOMNode to participate in the DOM tree structure and must implement the render() method to define their appearance.

Parameters:

Name Type Description Default

name

Optional display name for the widget. If not provided, generates a name using the class name and part of the ID.

 required

**kwargs

Additional widget properties: id: Unique identifier (auto-generated if not provided). x, y: Position coordinates (default: 0, 0). width, height: Dimensions (default: 0, 0). pos: Grid position for layout widgets (default: (0, 0)).

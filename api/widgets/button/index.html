Button 

Button(
    label,
    style="solid default icon",
    *,
    disabled=False,
    padding=(0, 0, 0, 0),
    on_click=None,
    state="default",
    **kwargs
)

Bases: Widget

A clickable button widget with customizable appearance and behavior.

The Button widget supports various visual styles, colors, sizes, and states. It can display text and respond to mouse clicks with configurable callbacks.

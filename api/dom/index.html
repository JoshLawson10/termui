mark_dirty 

mark_dirty()

Mark this node as dirty, indicating it needs to be re-rendered.

mark_dirty_cascade_down 

mark_dirty_cascade_down()

Mark this node and all its descendants as dirty.

Propagates the dirty flag down the DOM tree to ensure all child nodes are re-rendered when a parent changes.

mark_dirty_cascade_up 

mark_dirty_cascade_up()

Mark this node and all its ancestors as dirty.

Propagates the dirty flag up the DOM tree to ensure parent nodes are also re-rendered when a child changes.

DOMTree 

DOMTree()

A DOM tree for managing widget hierarchies and rendering order.

The DOMTree maintains a hierarchical structure of widgets and provides methods for traversal, manipulation, and debugging.

termui.color

Color dataclass 

Color(r, g, b, a=255)

A class to represent a color with RGBA values.

Parameters:

Name Type Description Default

r

int

Red component (0-255).

 required

g

int

Green component (0-255).

 required

b

int

Blue component (0-255).

 required

a

int

Alpha component (0-255). Defaults to 255 (fully opaque).

 255

inverse property 

inverse

Get the inverse color.

Returns:

Type Description
Color

A new Color instance with inverted RGB values (255 - component).
Color

Alpha value remains unchanged.

monochrome property 

monochrome

Convert the color to a monochrome (grayscale) color.

Uses the standard luminance formula: 0.2126R + 0.7152G + 0.0722*B

Returns:

Type Description
Color

A new Color instance with grayscale values. Alpha remains unchanged.

rgb property 

rgb

Convert the color to an RGB tuple.

Returns:

Type Description
tuple[int, int, int]

A tuple containing (r, g, b) values.

rgba property 

rgba

Convert the color to an RGBA tuple.

Returns:

Type Description
tuple[int, int, int, int]

A tuple containing (r, g, b, a) values.

 required

colorize 

colorize(text, fg=None, bg=None)

Colorize text with ANSI escape codes.

Parameters:

Name Type Description Default

text

str

The text to colorize.

 required

fg

Optional[Color]

The foreground color. If None, no foreground color is applied.

 None

bg

Optional[Color]

The background color. If None, no background color is applied.

 None

Returns:

Type Description
str

The text wrapped with appropriate ANSI escape codes for coloring.
str

If both fg and bg are None, returns the original text unchanged.

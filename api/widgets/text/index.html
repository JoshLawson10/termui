Text 

Text(
    content,
    fg_color=Color(255, 255, 255),
    bg_color=None,
    align="left",
    **kwargs
)

Bases: Widget

A widget that displays text.

Parameters:

Name Type Description Default

content

str | list[str]

The text content to display. Single strings become one line, lists of strings become multiple lines.

 required

fg_color

Color

The foreground color of the text (default: white).

 Color(255, 255, 255)

bg_color

Optional[Color]

The background color of the text (default: None).

 None

align

HorizontalAlignment

The alignment of the text (default: left).

 'left'

**kwargs

Additional widget configuration options.

 {}

How the content should be aligned horizontally.

get_minimum_size 

get_minimum_size()

Get the minimum size required to display the text content.

Calculates the smallest possible dimensions that can accommodate all text without truncation.

Returns:

Type Description
tuple[int, int]

tuple[int, int]: Minimum (width, height) in characters.

set_content(content)

Update the text content and recalculate widget size.

Changes the displayed text content and automatically adjusts the widget dimensions to fit the new content. Marks the widget as dirty to trigger re-rendering.

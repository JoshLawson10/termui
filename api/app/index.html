This is the base class for all TermUI applications

Parameters:

Name Type Description Default

screen

Screen

The screen instance to register.

 required

Raises:

Type Description

run 

run()

Run the application synchronously.

A convenience method that wraps run_async() with asyncio.run(). Handles keyboard interrupts and other exceptions gracefully.

Parameters:

Name Type Description Default

screen_name

str

The name of the screen to switch to.

 required

Raises:

Type Description
ScreenError

If the screen name is not found in registered screens.
